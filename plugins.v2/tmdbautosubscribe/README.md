# TMDB 自动订阅

根据 TMDB 数据发现近期内容，并生成 MoviePilot 订阅建议。插件默认只生成建议，只有开启“自动订阅”后才会提交到 MoviePilot。

## 能发现什么

| 类型 | 判断方式 | 订阅粒度 |
| --- | --- | --- |
| 新上映电影 | `primary_release_date` 落在扫描窗口 | 电影 |
| 新剧首播 | `first_air_date` 落在扫描窗口 | 剧集第 1 季 |
| 老剧新季 | 剧集详情中的 `season.air_date` 落在扫描窗口 | 对应季 |

同一季的后续新集只会进入“缓存观察”，不会自动生成订阅。这样可以避免长期连载剧每次播出都重复触发订阅。

## 首次使用

1. 在 MoviePilot 插件市场安装 `TMDB自动订阅`。
2. 保持“自动订阅”关闭，先使用“立即扫描”检查候选结果。
3. 设置扫描时间和媒体类型；默认是“未来 7 天、剧集、动画、日本、日语”。
4. 按需展开“细节分类”，设置类型、原产国、原始语言、排除类型和黑名单。
5. 在详情页检查“订阅建议”和“候选明细”。确认结果符合预期后，再开启自动订阅或插件定时任务。

`TMDB API Key` 留空时使用 MoviePilot 的全局 `TMDB_API_KEY`。如未配置全局 Key，也可以在插件中单独填写。

## 配置说明

### 基础配置

- `启用插件`：启用后按照“执行周期”定时扫描。
- `立即运行一次`：保存配置后执行一次扫描，随后自动复位。
- `自动订阅`：关闭时只生成建议；开启时才向 MoviePilot 提交符合规则的项目。
- `启动通知`：扫描完成后发送候选数量和建议数量，不包含明细。
- `跳过非中文标题`：默认开启。TMDB 返回的显示标题不含汉字时，只保留在候选明细，不生成建议；关闭后按其余规则继续处理。
- `过去扫描天数` / `未来扫描天数`：定义扫描日期窗口。默认只扫描未来 7 天。
- `媒体类型`：可选择电影、剧集或两者。

### 细节分类

- `电影类型`、`剧集类型`：TMDB 原生类型，多选为任一匹配。
- `排除电影类型`、`排除剧集类型`：使用 TMDB 原生排除条件。
- `原产国`、`原始语言`：多选为任一匹配；不选择表示不限。
- `黑名单`：填写 TMDB ID 后直接跳过。支持 `123`、`movie:123`、`tv:123` 和 `all:123`，可用逗号或换行分隔。

### 扫描参数

- `TMDB排序方式`：只能选择“按热度”或“按日期”。按热度优先热门条目；按日期优先窗口内较新的内容。
- `电影/新剧页数`：电影和新剧发现队列固定拉取的页数。
- `播出最少页数` / `播出最多页数`：老剧新季队列的自适应翻页范围。
- `低新连续页数` / `每页有效新数据`：控制老剧新季队列何时停止继续翻页。
- `TMDB超时秒数` / `TMDB重试次数`：单个 TMDB 请求的等待和重试限制。
- `清空缓存`：下一次初始化时清空最近扫描结果、筛选缓存和剧集观察缓存。

## 订阅保护

开启自动订阅时，插件会先读取 MoviePilot 已有订阅：

- 电影按“电影 + TMDB ID”去重。
- 剧集按“剧集 + TMDB ID + 季号”精确去重。
- 同一轮扫描中已经成功提交的项目不会重复提交。
- 读取订阅列表失败时，本轮自动订阅直接停止，不会盲目提交。

## 代理与网络

插件没有独立代理开关，始终跟随 MoviePilot 系统代理配置：

- 配置 `PROXY_HOST` 时使用该代理。
- 未配置 `PROXY_HOST` 时使用容器的 `HTTPS_PROXY` 或 `HTTP_PROXY`。
- 都未配置时直接访问 TMDB。

TMDB 网络异常会显示在扫描结果和日志中。可先确认全局 TMDB Key、TMDB API 域名和系统代理是否可用。

## 结果页面

- `订阅建议`：符合规则、可提交订阅的项目。
- `候选明细`：本次从 TMDB 拉取并分析过的所有项目，可用于排查筛选与漏扫。
- `拉取诊断`：显示每个队列拉取页数、总量和停止原因。
- `缓存观察`：记录已见过剧集的季、集数和最近播出信息变化；仅用于观察，不会订阅同季新集。

## 已知边界

- TMDB 资料延迟或缺少 `season.air_date` 时，老剧新季可能无法在首播前发现。
- 同一季分多个时间段播出的作品，在 TMDB 仍被定义为同一季时，不会因后续新集自动订阅。
- 排序只影响 TMDB 拉取顺序，不会改变日期窗口和筛选条件。
- MoviePilot 自身的订阅、识别和下载规则仍由 MoviePilot 负责处理。

## 反馈问题

提交 Issue 时请提供 MoviePilot 版本、插件版本、扫描窗口、筛选条件、相关队列诊断和报错日志。请勿附上 TMDB API Key、Cookie 或其他敏感信息。
